Ankify

Ankify is a lightweight command-line program used to format .txt files into a format for direct import into Anki. Ankify was developed using C++ and resolved a personal bottleneck in study habits i.e. efficiently creating flashcards from question and answer notes.

The current repo also includes prototype code for integrating Ankify with openAI to generate flashcards around any chosen topic, and to generate questions and answers pairs from prose notes. But this will be separated into another project.

The GitHub for Ankify can be found at github.com/l-hepper/ankify

Tech Stack: C++
